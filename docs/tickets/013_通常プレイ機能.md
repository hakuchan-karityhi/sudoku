# 013 通常プレイ機能

## 概要
通常プレイ（難易度選択）のパズル生成・取得機能をMVPで実装する。

## 詳細要件
- 難易度選択（初級・中級）
- 選択された難易度のパズル取得
- テンプレートベースまたは簡易生成ロジック
- スクリプト適用

## 実装ポイント
- getByDifficulty()メソッドの実装
- 難易度別パズルテンプレート準備
- ランダム生成または固定テンプレート使用
- MVPでは簡易アルゴリズム
