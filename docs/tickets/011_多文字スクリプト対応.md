# 011 多文字スクリプト対応

## 概要
漢字・アラビア文字・ローマ字での数字表示とRTL対応を実装する。

## 詳細要件
- 数字1-9のスクリプト別マッピング
- Script enumに応じた文字変換
- アラビア語のRTL（右→左）表示対応
- フォントレンダリングの品質確保

## 実装ポイント
- スクリプト変換ユーティリティ作成
- Directionality.rtlの適用
- Notoフォントの適切な使用
- 文字崩れ防止のための画像ベース検討（オプション）
