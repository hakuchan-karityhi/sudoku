# 012 デイリーパズル機能

## 概要
デイリーパズルの取得・表示機能をMVP（擬似デイリー）で実装する。

## 詳細要件
- 本日分のJSONデータ取得（assets/puzzles/YYYY-MM-DD.json）
- 未配信日のフォールバック処理
- デイリーパズルの公平性確保（端末生成禁止）
- 練習モード表示（未配信時）

## 実装ポイント
- assets/puzzles/ 配下にJSON配置
- LocalDataSourceを使用
- 日付フォーマット（yyyy-MM-dd）
- エラーハンドリング（ファイル不存在時）
