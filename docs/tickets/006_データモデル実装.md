# 006 データモデル実装

## 概要
Data層のDTO（PuzzleDto）とEntity変換ロジックを実装する。

## 詳細要件
- PuzzleDtoクラス：JSONからの変換用データモデル
- fromJson()：JSON → PuzzleDto変換
- toEntity()：PuzzleDto → Board変換
- 日付・難易度・グリッド・スクリプトのマッピング

## 実装ポイント
- lib/core/data/models/ に配置
- JSON構造：{date, difficulty, grid[9][9], script}
- enum変換：string → Difficulty/Script
- グリッド変換：int配列 → Cellリスト
