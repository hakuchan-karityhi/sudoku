# 017 パズルデータ準備

## 概要
MVP用のパズルJSONデータをassets/puzzles/ に準備する。

## 詳細要件
- 複数日のデイリーパズルJSON作成
- 各難易度のパズルデータ
- 有効な数独パズルデータ
- スクリプト情報付与

## 実装ポイント
- assets/puzzles/YYYY-MM-DD.json 形式
- 既存の有効パズルを使用
- 難易度分類
- スクリプトランダム割り当て
